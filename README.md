## Projects

### [Corpus](https://github.com/eren-o23/corpus) тАФ Grant Intelligence Pipeline
Agentic pipeline that crawls NIH Guide, Grants.gov, and private foundations via
Bright Data, matches opportunities to a lab profile using pgvector semantic search,
and drafts a full Letter of Intent via the Claude API тАФ end to end in one loop.

`Claude API` `pgvector` `Supabase Realtime` `Next.js` `Bright Data` `Resend`

---

### [DSAR LangGraph Agent](https://github.com/eren-o23/DSAR-LangGraph-Agent) тАФ Compliance Automation
LangGraph state machine automating DSAR triage: classification, scoping, risk
flagging, and a human-in-the-loop checkpoint before any decision is finalised.
LLM + deterministic fallback hybrid тАФ structured fallback warnings, no silent failures.
Episodic memory stores approved decisions as precedents for future requests.

`LangGraph` `LangChain` `Pydantic` `Ollama / OpenAI`

---

### [ML Pharmacy Demand Forecasting](https://github.com/eren-o23/ML-Pharmacy-Demand-Forecasting)
Full-stack forecasting system using LightGBM quantile regression тАФ data ingestion
through to a live dashboard.

`LightGBM` `Python` `PostgreSQL` `Next.js`
